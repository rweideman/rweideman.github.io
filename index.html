Projects

Mining Dashboard

The capstone to my internship at SSR Mining included making a fully operational geospatial dashboard that actively tracked the positions of all company drills at the Marigold Mine Site.

The map was interactive, showing each drill as well as each drill’s activity over various time ranges set by the user.

The dashboard was built using data pipelining, using SQL and Lua to extract the tags from a database so they could be used by a Python script to actively pull the data and plot the most recent positioning.

Arm Torque Tracker

The Arm Torque Tracker was a project I developed and tested along with a small cohort of biomedical engineering students that tracked a thrower’s arm throughout the full range of motion, allowing users to monitor internal muscle signaling and reduce strain-based injuries.

While on the electrical subteam, I acted as the Electrical Lead and contributed through PCB design, circuit load testing, board pin mapping, and data processing.
